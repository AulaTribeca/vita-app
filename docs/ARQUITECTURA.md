# Arquitectura funcional

Tablas principales:

- `profiles`
- `households`
- `household_members`
- `vita_one_items`
- `vita_one_shopping_items`
- `vita_one_relations`
- `vita_push_subscriptions`

La app parte de ciclos:

- cita médica → preparación → cierre → documento → seguimiento;
- medicación → stock → compra → reposición;
- UNED → asignatura → examen/entrega → próximo paso;
- compra → producto por línea → checkbox → tachado;
- hogar → dinero/coche/contactos/wallet.
