# Roadmap de VITA

## Nivel 1, base visual
Objetivo: crear una PWA navegable, clara y visual, sin datos sensibles reales.

## Nivel 2, Supabase y seguridad
Objetivo: autenticación, usuarios, perfiles, hogar compartido y Row Level Security.

## Nivel 3, registros esenciales
Objetivo: salud diaria, medicación, citas, volantes médicos, tareas y hogar.

## Nivel 4, documentos
Objetivo: subida de fotos/PDF, carpetas, estados y relación con citas.

## Nivel 5, recordatorios y avisos
Objetivo: avisos internos, recordatorios automáticos, checklist por tipo de evento y notificaciones push.

## Nivel 6, exportación
Objetivo: exportar PDF, CSV/JSON y ZIP completo con documentos.

## Nivel 7, perfil de Román y hogar compartido completo
Objetivo: datos privados por usuario y espacio común de pareja.
