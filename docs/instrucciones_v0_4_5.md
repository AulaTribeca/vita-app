# VITA v0.4.5

## Archivos que debes copiar encima

- `index.html`
- `app.js`
- `config.js`
- `service-worker.js`
- `README.md`
- `docs/instrucciones_v0_4_5.md`

## Después

1. Commit en GitHub Desktop:
   `VITA v0.4.5 corregir carga Supabase`
2. Push origin.
3. Espera el despliegue de GitHub Pages.
4. Abre VITA en ventana de incógnito.
5. Entra con:
   - Usuario: `Patricia`
   - Contraseña: la creada en Supabase

## Si sigue sin entrar

1. Cambia la contraseña del usuario en Supabase.
2. Revisa Authentication → Providers → Email.
3. Comprueba que Email/Password está activado.
4. Borra caché de la app o abre en incógnito.
