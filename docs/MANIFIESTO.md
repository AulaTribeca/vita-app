# Manifiesto VITA ONE

VITA ONE no es una agenda. No es una lista. No es una base de datos bonita.

VITA ONE convierte la vida dispersa en próximos pasos claros.

Su promesa es sencilla:

- mostrar lo que toca ahora;
- sostener los ciclos importantes;
- reducir decisiones;
- acompañar sin invadir;
- hacer bello lo cotidiano.

La app trabaja para la persona, no la persona para la app.
