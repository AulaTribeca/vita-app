# VITA v0.5, instalación limpia

## Pasos

1. Cierra la app en el navegador.
2. Entra en la carpeta local del repositorio.
3. Borra todo el contenido de la carpeta.
4. Descomprime `vita_v0_5_completa.zip`.
5. Copia dentro del repositorio todo su contenido:
   - `assets`
   - `docs`
   - `app.js`
   - `config.js`
   - `index.html`
   - `manifest.webmanifest`
   - `README.md`
   - `service-worker.js`
   - `styles.css`
6. Abre GitHub Desktop.
7. Commit:
   `VITA v0.5 carpeta completa`
8. Push origin.
9. Espera a que GitHub Pages despliegue.
10. Abre VITA en ventana de incógnito.

## Si no entra

1. Revisa que el usuario tenga contraseña en Supabase.
2. Revisa Authentication → Sign In / Providers → Email.
3. Confirma que email/password está activado.
4. Abre en incógnito.
5. Borra service worker anterior si fuese necesario.
