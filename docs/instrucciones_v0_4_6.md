# VITA v0.4.6

## Archivos que debes copiar encima

- `index.html`
- `app.js`
- `config.js`
- `service-worker.js`
- `README.md`
- `docs/instrucciones_v0_4_6.md`

## Después

1. Commit en GitHub Desktop:
   `VITA v0.4.6 login REST`
2. Push origin.
3. Espera el despliegue.
4. Abre VITA en incógnito.
5. Entra con:
   - Usuario: `Patricia`
   - Contraseña: la nueva contraseña creada en Supabase.

## Importante

Cambia la contraseña que apareció en la captura antes de seguir.

## Si no entra

Revisa en Supabase:

Authentication → Sign In / Providers → Email

Debe estar activado el acceso por email/contraseña.

También revisa que el usuario Patricia tenga contraseña válida.
