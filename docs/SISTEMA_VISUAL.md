# Sistema visual helénico

Principios:

- mármol cálido, travertino, olivo, bronce y terracota;
- tarjetas como placas;
- botones como sellos;
- separadores inspirados en frisos;
- iconografía lineal, no emojis;
- movimiento suave;
- ausencia de mensajes técnicos en la interfaz.
